The program assigns hotkeys for:
* Changing the keyboard case with a single key (Caps Lock)
* Displaying an email address (Ctrl+F3)
* Displaying a phone number (Ctrl+F4)
* Launching the program (Ctrl+F5)
* Displaying a signature (Ctrl+F9)
* Changing the case of selected text anywhere on the screen (scr_lk).

To change the case of selected text, press the hotkey (scr_lk).
After wards, review the suggested replacement in the dialog box
and decide whether to use it or not.
When using the program with the '--fast' parameter,
the dialog box does not appear; the replacement always occurs.

Programming language: Python.
