The program assigns hot keys for:
* Changing the keyboard case with one key (Caps Lock)
* Displaying an e-mail address (Ctrl+F3)
* Displaying a phone number (Ctrl+F4)
* Launching the program (Ctrl+F5)
* Displaying a signature (Ctrl+F9)
* Changing the keyboard case in the selected text, anywhere on the screen (scr_lk).
To change the case in the selected text, you need to press the hot key (scr_lk).
After that, you need to view the proposed replacement option in the dialog box
and decide whether to use it or not.
Programming language — Python.
